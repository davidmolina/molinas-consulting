<p class="intro"><span class="dropcap">C</span>omo se organiza una campana politica con recursos financieros limitados? Integrando herramientas open source y una operacion descentralizada.</p>

# Desafio
El cliente competia en una campana politica con presupuesto reducido. Debido a la amplitud geografica del distrito en disputa, coordinar personas en un solo lugar resultaba costoso y consumia demasiado tiempo operativo.

# Solucion
Trabajamos con el candidato y su campaign manager para disenar una estrategia remota/descentralizada, apalancando APIs y herramientas open source para ejecutar una campana competitiva dentro de un presupuesto bootstrap.

# Resultados
Al cierre del acompanamiento, el cliente contaba con una hoja de ruta politica bien documentada, herramientas open source implementadas y una oficina remota/descentralizada para coordinar voluntarios y equipo de campana con mayor efectividad.
