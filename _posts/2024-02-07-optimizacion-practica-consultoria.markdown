<p class="intro"><span class="dropcap">C</span>omo se escala una pequena practica de consultoria sin infraestructura fisica? Adoptando herramientas open source y descentralizando la operacion.</p>

# Desafio
El cliente necesitaba una herramienta de comunicacion para evitar cuellos de botella en correo electronico, acelerar la ideacion y mejorar la coordinacion interna en la entrega de servicios.

# Solucion
Trabajamos con el socio director para disenar una estrategia de operacion remota y descentralizada, apalancando APIs y herramientas open source para coordinar, integrar y sincronizar la practica consultiva, incluso en un esquema bootstrap o de presupuesto cero.

# Resultados
Al cierre del acompanamiento, el cliente contaba con una hoja de ruta estrategica documentada y una oficina remota/descentralizada para coordinar consultores asociados y escalar con mayor efectividad. Esto incluyo un formulario de intake para prospectos conectado directamente a la aplicacion de comunicacion para mejorar tiempos de respuesta.
